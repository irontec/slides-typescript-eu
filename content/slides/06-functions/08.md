## Arrow functions (II)

Arrow functionsak erabiliz, **this**-ek Person motako objetua izaten jarraituko du.

```typescript
class Person {
  name = 'Juan';
  sleep(): Promise<string> {
    return new Promise((resolve) => {
      resolve(`wake up ${this.name}`);
    });
  }
}
const p = new Person();
p.sleep().then(console.log); // wake up Juan
```
