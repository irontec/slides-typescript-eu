## Arrow functions

_arrow functions_-en erabilerak exekuzio testuingurua mantentzen uzten du.

```typescript
class Person {
  name = 'Juan';
  sleep(): Promise<string> {
    return new Promise(function (resolve) {
      resolve(`wake up ${this.name}`);
    });
  }
}

const p = new Person();
p.sleep().then(console.log); // wake up undefined
```
