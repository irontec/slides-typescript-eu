## any

Any motarekin, konpilazio garaian type-checkerra saihesten dugu. JS klasikoan bezala, tipatu ahula izatea ahalbidetzen digu:
```typescript
let a : any = true;
a = 'Ahora un string!';
```

Era berean, erabilgarria da hainbat elementuko arraya osatzeko:
```typescript
let a:any[] = [];
a.push(12);
a.push('Car');
```
