### array

Array bat izango duen datu mota erabiliz adierazi daiteke:

```typescript
let names: string[] = [];
names.push('Jon');
names.push('Ane');
```

Edo Array mota orokorra erabiliz, bere elementuen mota zehaztuz:

```typescript
let names: Array<string> = [];
names.push('Jon');
names.push('Ane');
```
